# NotebookLM Source Management

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-2.2.0-green.svg)

> **NotebookLM Source Management** is a powerful Chrome extension that supercharges your Google NotebookLM experience with advanced source management, batch operations and so on.
> 
> **NotebookLM Source Management** 是一款强大的 Chrome 浏览器扩展，为 Google NotebookLM 带来来源管理功能、批量删除操作等等。

---

## 🌟 Highlights / 核心亮点

- 🗑️ **One-Click Batch Delete (一键极速批量删除)**: Tired of clicking "Delete" on sources one by one? Enter the focused deletion mode and batch delete multiple sources in milliseconds! 
- 🎨 **Glassmorphism UI (极致的毛玻璃美学)**: Apple HIG-inspired design with `12px` rounded corners, ultra-smooth `cubic-bezier` animations, and global translucent elements.
- 🌍 **Native Dual-Language Support (原生双语支持)**: Built-in internationalization (`en` and `zh_CN`). The UI automatically adapts to your browser's language setting.
- 📂 **Smart Folder Animations (丝滑的文件夹动效)**: Effortlessly organize your sources with new folder lifecycle entry animations.
- 🛡️ **Robust State Management (极度稳定的状态管理)**: Powered by Manifest V3 Background Service Workers to keep your source selection states safe and persistent.

## 🚀 Installation / 安装指南

### English
1. Download or clone this repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/notebooklm-source-plus.git
   ```
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top right corner.
4. Click **"Load unpacked"** and select the extension folder (`notebooklm-source-plus-main`).
5. Open [NotebookLM](https://notebooklm.google.com/) and enjoy!

### 简体中文
1. 下载或克隆本仓库到你的本地电脑：
   ```bash
   git clone https://github.com/yourusername/notebooklm-source-plus.git
   ```
2. 打开 Google Chrome 浏览器并输入地址 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式”** (Developer mode)。
4. 点击 **“加载已解压的扩展程序”** (Load unpacked)，然后选择你刚刚下载的插件文件夹（`notebooklm-source-plus-main`）。
5. 打开 [NotebookLM](https://notebooklm.google.com/)，开始体验极致的管理功能！
